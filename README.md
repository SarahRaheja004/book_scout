Book Scout : 
1. Aggreagtes textbook prices from multiple sources like Amazon, ecampus, CampusBooks etc
2. Users can enter an ISBN or a textbook title 
3. It displays the prices from all sources, sorted by the lowest price with links to buy or rent 


Tech stack used: 
Backend : Node.js
Frontend : React
Databse: MongoBD
Caching: Redis 
Deployment: Render
